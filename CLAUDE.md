# CLAUDE.md - Loopborn Saga

## Project Overview
The Loopborn Saga (also called the Chuckles saga) is a multi-part space opera by Mike.
This repo contains all worldbuilding, character development, story drafts,
audiobook production assets, and automation scripts.

## Current Status
- Part One: Complete
- Part Two: Complete
- Part Three: IN PROGRESS - full space sci-fi action phase

## Key Lore
- The Velkanni Protocol: central cosmic mechanism
- The Kairospan Artifact: key plot device
- The Nathiri Realm: major location/dimension
- MYDO = created beings, MIDO = human Earth versions

## Character Pronouns
- Chuckles, Rocky, Dreya, Elara, Mama, Mack, Anna: she/her
- Wren, Radek, Goose, Muck, James, Bubby: he/him
- Kiru: they/them

## Conventions
- Soul profiles: 02-CHARACTERS/soul-profiles/ one .md per character
- Chapters: 03-STORY/part-XX/chapters/ named ch-01.md, ch-02.md
- Image prompts: 05-ASSETS/image-prompts/
- Reusable Claude prompts: 06-AUTOMATION/prompts/
